# Valentine's Day Website üíï

A cute and interactive Valentine's Day website to ask someone special out!

## Features
- Interactive "Yes/No" buttons with fun animations
- The "No" button runs away when you try to click it
- Sweet celebration page when "Yes" is clicked
- Fully responsive design for mobile and desktop

## Deploy to Vercel

### Quick Deploy
1. Push this code to a GitHub repository
2. Go to [Vercel](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Click "Deploy"

That's it! Vercel will automatically detect and deploy your site.

### Manual Deploy (using Vercel CLI)
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

## Files
- `index.html` - Main landing page (entry point)
- `forher.html` - Original main page (kept for reference)
- `yes.html` - Success/celebration page
- `forher.css` - Styles for main page
- `yes_style.css` - Styles for success page
- `forher.js` - JavaScript for alert message
- `vercel.json` - Vercel configuration

## Local Development
Simply open `index.html` in your browser to test locally.
